## Requirements

To use the Latitude CLI, you need to have `Node.js >18.x` installed on your machine. You can download it from the [official website](https://nodejs.org/).

If you're using Windows, please refer to [Windows compatibility](./windows-compatibility) before installing.

## Installation

Run the following command to install the Latitude CLI globally:

<CodeGroup>
```bash npm
npm install -g @latitude-data/cli
```
```bash yarn
yarn global add @latitude-data/cli
```
```bash pnpm
pnpm add -g @latitude-data/cli
```
</CodeGroup>
