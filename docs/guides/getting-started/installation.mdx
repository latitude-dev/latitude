## Requirements

To use the Latitude CLI, you need to have Node.js installed on your machine. You can download it from the [official website](https://nodejs.org/).

## Installation

Run the following command to install the Latitude CLI globally:

<CodeGroup>
```bash npm
npm install -g @latitude-sdk/cli
```
```bash yarn
yarn global add @latitude-sdk/cli
```
```bash pnpm
pnpm add -g @latitude-sdk/cli
```
</CodeGroup>
