## Introduction
A mixed chart combines different chart types (for example, bars and lines) to present complementary data and give more context in a single visualization.

## Syntax
You can use `<MixedChart ... />` to add it in your view. Within the tag you must add the properties to configure the chart.

``` js
<MixedChart
  query='users_evolution'
  title="Users Evolution"
  description="Evolution of users in the last 12 months."
  x='created_at'
  y={[
    {name: 'count_users', chartType: 'bar'},
    {name: 'avg_users', chartType: 'line'},
    {name: 'sum_users', chartType: 'area'}
  ]}
  sort='count_users'
  xTitle='Month'
  yTitle='Users'
  config={{
    // ...
  }}
/>
```
