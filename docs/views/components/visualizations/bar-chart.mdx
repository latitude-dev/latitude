## Introduction
The Bar chart represents quantitative data with vertical or horizontal bars, showing comparisons across categories or time. It highlights differences in values, making it ideal for comparing multiple datasets and identifying trends or outliers.

## Syntax
You can use `<BarChart ... />` to add it in your view. Within the tag you must add the properties to configure the chart.

``` js
<BarChart
	query='sales_evolution'
    title='Sales Evolution'
    description='Monthly sales evolution'
    x='created_at'
    y='sales'
    sort={{ column: 'sales', order: 'desc' }}
    xTitle='Month'
    yTitle='Sales'
    config={{
        ...
    }}
/>
```
