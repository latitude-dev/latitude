## Introduction
A pie chart visualizes data as proportional segments of a circle. It makes easier to see relative sizes of categories at a glance, ideal for highlighting distribution and composition.

## Preparation
The query `.sql` file that the Pie Chart accepts must contain only 2 columns, one for the labels and another one for the values.

```
| labels    | values   |
| Drinks    | 145      |
| Snacks    | 539      |
| Fruits    | 987      |
```

## Syntax
You can use `<PieChart ... />` to add the pie chart. Within the tag you must add the properties to configure the chart.

``` js
<PieChart
	query='users_evolution'

    config={{ 
        showValues: true,
        showLegend: true,
        showDecal: false
    }}
/>
```

### Properties

#### Mandatory
- `query`: The name of your `.sql` file from your chart will get the data. It's not necessary to include the whole path, just the name, because we look into all folders until find it.

### Optional
- `config`: Allow to add more configuration and customize your chart.
    - `showLabels`: (*boolean*) 
    - `showHole`: (*boolean*) 
    - `showTotalValue`: (*boolean*) 
    - `showLegend`: (*boolean*) Display the color reference of each serie displayed.
    - `showDecal`: (*boolean*) Add a visual pattern to each serie to help colorblind people distinct them.
